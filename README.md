# Financial-Markets-impact-on-NFTs
These files contain the codes and supplementary materials associated with our academic paper, “How Do Traditional Financial Markets Affect NFT Prices? Insights into Asset and Sentiment Impacts”. 
The SHAPAnalysis script provides the empirical analysis for the overall contribution analysis. Its structure is consistent with the other empirical modules—Stocks & Bonds, Cryptocurrencies & DeFi, Exchange Rates, and Sentiment Index.
To apply the analysis to any specific market category, you only need to modify the set of independent variables (X) in the script.
